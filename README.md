# androidStoreManagement

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8-blue.svg)](https://kotlinlang.org)
[![Android](https://img.shields.io/badge/Android-Studio-green.svg)](https://developer.android.com/studio)
[![Firebase](https://img.shields.io/badge/Firebase-Realtime%20DB-orange.svg)](https://firebase.google.com)

Aplicação minimalista desenvolvida para a disciplina de Desenvolvimento Mobile. Demonstra autenticação de utilizadores e sincronização de dados em tempo real usando **Firebase Authentication** e **Firebase Realtime Database**. Implementada em **Kotlin** com **Android Studio**.

---

## Visão geral
Aplicação exemplo para gerir produtos de uma loja (produtos, stock) com interface minimalista, focada em clareza e código didático. Ideal para apresentação académica e como ponto de partida para projetos maiores.

---

## Funcionalidades principais
- Registo e autenticação de utilizadores (email/password) — Firebase Auth
- Criação, leitura, atualização e remoção de itens em tempo real — Firebase Realtime Database
- Interface limpa e simples (minimal UI)
- Estrutura de projeto organizada (activities/fragments, models, utils)

---

## Requisitos
- Android Studio (recomendado versão recente)
- JDK 11+
- Conta Firebase (projeto criado com Realtime Database e Authentication ativados)
- Dispositivo Android ou AVD (Android 7.0+ recomendado)
